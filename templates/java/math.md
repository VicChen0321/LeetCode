# 🧮 數學計算 (Math Utilities)

```java
// 向上取整（例如計算頁數）
int pages = (n + k - 1) / k;

// 向下取整
int result = n / k;

// 四捨五入
int rounded = (int) Math.round(x);

// 取最大最小值
int maxVal = Math.max(a, b);
int minVal = Math.min(a, b);

// 絕對值
int absVal = Math.abs(num);

// 平方與平方根
int square = x * x;
double sqrt = Math.sqrt(x);

// 取模（確保正值）
int mod = ((x % m) + m) % m;

// 次方
double pow = Math.pow(a, b);
```
