# 143 Reorder List

**LeetCode Link:**
[143. Reorder List](https://leetcode.com/problems/reorder-list/)

**Difficulty:** Medium
**Tags:** Linked List, Two Pointers, Stack

---

## 題目描述

給定一個單向鏈表 `L0 → L1 → … → Ln-1 → Ln`，請重新排序，使其變成：

```
L0 → Ln → L1 → Ln-1 → L2 → Ln-2 → …
```

不能修改節點的值，只能重新調整節點指向 (`next`)。

---

### 範例

**Input**

```
1 → 2 → 3 → 4
```

**Output**

```
1 → 4 → 2 → 3
```

---

**Input**

```
1 → 2 → 3 → 4 → 5
```

**Output**

```
1 → 5 → 2 → 4 → 3
```

---

## 解法概念

### 方法：三步驟（快慢指針 + 反轉 + 合併）

1️⃣ **使用快慢指針找中點**

* 快指針一次走兩步，慢指針一次一步
* 最終 slow 停在中間節點

2️⃣ **反轉後半段鏈表**

* 從中點開始把後半段反轉

3️⃣ **從頭與反轉後半段交錯合併**

* 合併兩條鏈表（交錯方式）
* 全程不使用額外儲存空間（符合題意要求 O(1)）
